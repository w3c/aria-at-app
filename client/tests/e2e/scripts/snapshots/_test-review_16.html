    Test 1: Navigate forwards to a link in reading mode

    JAWS

    Instructions
    1. Configure JAWS with default settings. For help, read Configuring Screen Readers for Testing.
    2. Activate the "Open test page" button, which opens the example to test in a new window and runs a script that sets focus on a link before the 'W3C website' link
    3. With the reading cursor on the 'Navigate forwards from here' link, navigate to the 'W3C website' link.
    4. With the reading cursor on the 'Navigate forwards from here' link, navigate to the 'W3C website' link. Do this with each of the following commands or command sequences.
